# KeenKeeper

KeenKeeper is a friendship relationship tracker that helps you keep meaningful connections active.

## Core Features

- Home dashboard with friend status summary and category filters
- Friend detail page with quick check-in actions (call, text, video)
- Timeline page with interaction type filtering
- Stats page with pie-chart analytics
- Fallback and recovery routing for missing pages and invalid friend profiles

## Tech Stack

- React 19
- Vite 8
- Tailwind CSS
- React Router
- Recharts
- Lucide React
- React Hot Toast

## Local Development

1. Install dependencies:
	npm install
2. Start development server:
	npm run dev
3. Build for production:
	npm run build
4. Preview production build:
	npm run preview

## Deployment

- This project is configured for Vercel SPA routing with vercel.json.
- Push to GitHub and import the repository in Vercel to deploy.
